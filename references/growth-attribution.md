# 涨粉归因分析方法论

> 本文档定义了涨粉归因的分析框架和计算方法。

## 1. 归因模型

### 1.1) 核心公式

```
涨粉驱动力 = f(内容因素, 行为因素, 关系因素, 策略因素)
```

### 1.2) 四大因素权重参考

| 因素 | 默认权重 | 范围 | 说明 |
|------|---------|------|------|
| 内容因素 | 40% | 30-50% | 内容类型、质量、形式的影响 |
| 行为因素 | 25% | 15-35% | 发帖频率、时间、互动行为 |
| 关系因素 | 20% | 10-30% | 社交网络效应（被大V转发等） |
| 策略因素 | 15% | 10-25% | 固定栏目、活动、简介优化等 |

> **注意**：权重不是固定的，AI 应根据具体账号数据动态调整。

## 2. 分析维度详解

### 2.1) 内容因素分析

#### 高互动内容识别

```python
# 高互动帖判定标准
high_engagement = tweet where:
  likes > median_likes * 2
  OR retweets > median_retweets * 3
  OR (likes + retweets + replies * 2) > top_10_percentile
```

#### 内容类型 vs 涨粉效率

```
计算方法：
  对于每种内容类型 C：
    avg_engagement(C) = sum(likes + retweets*3 + replies*2) / count(C)
    relative_efficiency(C) = avg_engagement(C) / overall_avg_engagement
```

#### Thread vs 短推对比

```
Thread 涨粉效率 = Thread 平均互动 / 全部平均互动
短推涨粉效率 = 短推平均互动 / 全部平均互动

如果 Thread 涨粉效率 > 2.0，说明 Thread 是核心增长手段。
```

### 2.2) 行为因素分析

#### 发帖频率分析

```
日均发帖量 = 总推文数 / 分析天数
发帖频率变化 = 最近 30 天日均 / 之前日均

判断：
  > 1.5 倍 → 加速期（可能对应涨粉加速）
  0.7-1.5 倍 → 稳定期
  < 0.7 倍 → 减速期
```

#### 发帖时间分析

```
按小时统计发帖量和互动量：
  hour_stats[0..23] = {post_count, avg_engagement}

最佳发帖窗口 = avg_engagement TOP 3 小时
```

#### 互动行为分析

```
回复他人推文数 / 自发推文数 = 互动比
  > 2.0 → 高互动型（可能通过回复吸引关注）
  0.5-2.0 → 平衡型
  < 0.5 → 内容输出型
```

### 2.3) 关系因素分析

#### 被转发分析

```
识别推文被谁转发了：
  - 是否有大 V (followers > 10K) 转发？
  - 转发后的互动增量是多少？
  - 被转发帖的内容类型是什么？
```

#### 社交网络效应

```
通过 followers 和 following 分析：
  followers/following ratio：
    > 5.0 → 意见领袖型
    1.0-5.0 → 增长型
    < 1.0 → 社交型（大量关注别人）
```

### 2.4) 策略因素分析

#### 固定栏目/系列识别

```
识别方法：
  1. 查找重复出现的格式（如 "Day X of Y"、"每周一"、"Thread 🧵"）
  2. 查找重复出现的话题标签
  3. 查找重复出现的内容结构
```

#### 简介变化追踪

```
如果能获取简介历史变化（通常需要多次采样）：
  - 简介是否包含明确的 CTA？
  - 是否有链接引流？
  - 是否有身份标签/关键词优化？
```

## 3. 归因输出格式

### 3.1) 标准输出

```
📈 涨粉归因分析

账号阶段判断：[起步期/增长期/爆发期/稳定期]
当前增长速度：[缓慢/中等/快速/极快]

核心驱动因素（按权重）：
  1. 🎯 [XX%] 因素名称 → 简要说明
     证据：具体数据支持

  2. 💬 [XX%] 因素名称 → 简要说明
     证据：具体数据支持

  3. ⏰ [XX%] 因素名称 → 简要说明
     证据：具体数据支持

关键转折点：
  - YYYY-MM-DD：发生了什么 → 带来了什么变化

风险因素：
  - 哪些做法不可持续或有风险
```

### 3.2) 快速版（只输出 TOP 3）

```
涨粉 TOP 3 原因：
  1. [最核心原因]
  2. [第二原因]
  3. [第三原因]
```

## 4. 常见涨粉模式识别

| 模式 | 特征 | 适用性 |
|------|------|--------|
| **干货输出型** | 高质量技术/教程帖为主，Thread 是核心武器 | 适合专业人士 |
| **观点领袖型** | 站队鲜明、观点犀利，靠争议传播 | 适合有独立见解的人 |
| **互动社交型** | 高频回复大V、积极参与讨论 | 适合新手起步 |
| **资源聚合型** | 整理/分享工具、资源、列表 | 适合信息整理能力强的 |
| **故事叙事型** | 个人经历/成长故事，情感共鸣 | 适合有故事的人 |
| **热点追踪型** | 快速跟进热点事件发表观点 | 适合反应快、视野广的 |
