# 内容分类参考标准

> 本文档定义了推文内容分类的标准维度和标签体系，供 AI 分析时参考。

## 1. 内容类型分类

### 1.1) 主类型标签

| 标签 | 定义 | 典型特征 | 涨粉效率 |
|------|------|----------|---------|
| `技术干货` | 分享具体技术知识、代码、工具 | 含代码片段/链接/教程步骤 | ⭐⭐⭐⭐⭐ |
| `行业观点` | 对行业趋势、事件的个人见解 | 立场明确、有论据支撑 | ⭐⭐⭐⭐ |
| `个人故事` | 个人经历、成长感悟 | 第一人称叙事、情感真实 | ⭐⭐⭐⭐ |
| `教程指南` | 步骤化的操作指南 | 1️⃣2️⃣3️⃣编号、实操性强 | ⭐⭐⭐⭐⭐ |
| `资源分享` | 分享有价值的工具/资源/链接 | "推荐"、"分享"、"发现" | ⭐⭐⭐ |
| `互动提问` | 向社区提问或发起讨论 | 问号结尾、"你们觉得" | ⭐⭐⭐ |
| `热点评论` | 对时事热点的快速点评 | 涉及 trending 话题 | ⭐⭐⭐⭐ |
| `产品营销` | 推广产品、服务、课程 | CTA 明显、"点击"、"注册" | ⭐⭐ |
| `日常感悟` | 非行业的个人日常 | 生活化、轻松 | ⭐⭐ |
| `段子幽默` | 搞笑、玩梗、段子 | 搞笑/讽刺/夸张 | ⭐⭐⭐ |
| `数据报告` | 分享数据分析结果 | 含数字/图表/统计 | ⭐⭐⭐⭐ |
| `长文深度` | Thread 形式的深度分析 | 多条连贯推文 | ⭐⭐⭐⭐⭐ |

### 1.2) 判断优先级

如果一条推文同时符合多个类型，按以下优先级选择主类型：

1. 长文深度（Thread 形式优先标记）
2. 教程指南
3. 技术干货
4. 行业观点
5. 数据报告
6. 资源分享
7. 热点评论
8. 个人故事
9. 互动提问
10. 产品营销
11. 段子幽默
12. 日常感悟

## 2. 内容形式分类

| 形式 | 特征 | 互动率参考 |
|------|------|-----------|
| `纯文字` | 无图片、无链接 | 基准 |
| `图片` | 含 1-4 张图片 | +30-50% |
| `视频` | 含视频附件 | +20-40% |
| `Thread` | 多条连贯推文 | +100-300% |
| `引用转发` | 引用他人推文并评论 | +10-20% |
| `投票` | Twitter Poll | +50-100% |
| `链接分享` | 含外部链接 | -10-20%（但引流效果好） |

## 3. 情绪基调分类

| 基调 | 特征 | 适用场景 |
|------|------|----------|
| `积极鼓励` | 正面、激励、乐观 | 个人品牌建设 |
| `中性客观` | 事实陈述、数据展示 | 专业性展示 |
| `争议挑战` | 质疑、反直觉、站队 | 引发讨论/传播 |
| `共情共鸣` | 理解、陪伴、"我也是" | 建立情感连接 |
| `好奇探索` | 提问、假设、尝试 | 引发互动 |
| `批评反思` | 指出问题、反思教训 | 树立专业形象 |

## 4. 互动钩子分类

| 钩子类型 | 定义 | 示例 | 涨粉效果 |
|---------|------|------|---------|
| `提问式` | 直接问问题 | "你们觉得 AI 会取代程序员吗？" | ⭐⭐⭐ |
| `观点站队` | 明确立场激化讨论 | "说个得罪人的话：90% 的敏捷都是形式主义" | ⭐⭐⭐⭐⭐ |
| `资源诱惑` | 承诺提供有价值的资源 | "整理了 50 个 AI 工具，需要的扣 1" | ⭐⭐⭐⭐ |
| `悬念设计` | 先给结论再展开 | "我花了 3 个月才搞明白的一件事…" | ⭐⭐⭐⭐ |
| `共情触发` | 触发读者情感共鸣 | "35 岁转行做程序员，第一天上班的感受" | ⭐⭐⭐⭐ |
| `数据冲击` | 用数字震撼读者 | "这个工具帮我省了 200 小时/年" | ⭐⭐⭐⭐ |

## 5. AI 分类 Prompt 模板

在对推文进行分类时，AI 应使用以下思考框架：

```
分析这条推文：

内容：[推文全文]
互动数据：likes=X, retweets=X, replies=X

请判断：
1. 主内容类型（从 12 种中选择 1 个最匹配的）
2. 内容形式（纯文字/图片/视频/Thread/…）
3. 情绪基调（从 6 种中选择 1 个）
4. 互动钩子（如有，从 6 种中选择）
5. 为什么这条推文互动高/低？（1 句话）
```
